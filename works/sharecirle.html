Share Circle
Version 1.0

Share Circle is the five-click solution for creating any list of icons like for example the social networks' icons. On your website it will use less space and at the same time look much nicer thanks to lots of customization options. The integration is really simple.

Usage

Integration

There are two ways to integrate Share Circle - LESS and CSS. Each of them has its pros and cons. When i developed the script I expected that users will set up the script using LESS and than final result "will be converted" to CSS file which will be connected to a web page.

I highly recommend to follow it because if you want to use script with Internet Explorer it won't work correctly. In other browsers it works well but is loading much slower than CSS-version. Only one plus which is on side LESS it is ability to make on-the-fly changes. In that case you can keep LESS file on you computer or create a new one in 5 minutes.

Follow next 4 steps to get Share Circle working
1.
Open .LESS file and change parametres as you like
2.
Generate .CSS from that .LESS file on http://less2css.org/ and paste it to styles.css
3.
Remove links to less.js and styles.less from index.html and keep link just to styles.css and jquery.js
4.
Well done!

Warning!

Less.js currently won’t work if you’re using Chrome and script is located in your computer folder. In that case will be loaded default style.css instead of style.less. You can easily recognise if it happens - Share Circle will be in red colors but still working. To fix it just open index.html in another browser or upload Share Circle on the server.

API referance

The main plus of the LESS files is ability to make changes in one parametre and it will be changed in whole genereated CSS file. There are 9 different paremetres which can be changed. Their amount will be increased in future. Description each of parametre is below.

Option Default value Description
@size 40px 20%
@radius 80px Yes
@border-radius 20px No
@count 4 No
@speed 300 Yes
@showLabel false yes
@align left No
@color #ffffff No
@gradientColor1 #5126a2 No
@gradientColor2 #7558cc No
